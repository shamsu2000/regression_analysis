# Marketing ROI Analysis using Simple Linear Regression

## Project Overview
This repository contains a complete data science workflow utilizing Python and `statsmodels` to build a Simple Linear Regression model. The objective of this project is to analyze marketing dataset parameters (TV, Radio, Social Media) to determine which channel drives the highest sales and to formulate a clear, ROI-based recommendation for marketing budget allocation. 

Key steps included in this analysis:
* Data loading, exploration, and cleaning (handling missing values).
* Exploratory Data Analysis (EDA) and relationship visualization.
* OLS regression model implementation.
* Rigorous diagnostic testing (Linearity, Normality, and Homoscedasticity).
* Translation of statistical outputs (R-squared, p-values, coefficients) into actionable business strategy.

## Key Findings
* **Winning Channel:** TV Advertising demonstrated the strongest correlation with overall sales.
* **Model Accuracy:** The validated OLS model yielded an $R^2$ of 0.999, indicating a highly predictable relationship.
* **Business ROI:** The model's coefficient reveals that for every additional 1 unit of currency spent on TV advertising, the business can expect a reliable return of ~3.56 units in sales.
